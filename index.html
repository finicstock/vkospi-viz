VKOSPI = 가중 곡선 아래 '넓이'

공식의 합(Σ)은 아래 곡선 넓이의 리만합 근사 — 즉 VKOSPI² ≈ 색칠된 면적입니다. 변동성을 0까지 내려 넓이가 사라지는 극한도 확인해 보세요.

① 원재료: OTM 옵션 중간호가 Q(K)
K₀ 왼쪽은 풋, 오른쪽은 콜 가격 — ATM 근처에서 가장 비싸고 양쪽 끝으로 갈수록 싸짐
↓  여기에 2erT / (T·K²) 가중을 곱하면 — 낮은 행사가일수록 크게 증폭 ↓
② 가중 곡선 f(K) — 이 곡선 아래 넓이가 곧 σ² (30일 분산)
공식의 Σ ΔK·erTQ(K)/K² = 세로 막대들의 넓이 합 = 색칠 면적의 근사

변동성 0 → 모든 OTM 옵션이 무가치 → 곡선이 바닥에 붙고 넓이 = 0 → VKOSPI = 0. 옵션 프리미엄이 존재하는 한 VKOSPI는 0이 될 수 없다는 뜻이기도 합니다.

실험 제안: ① 변동성을 0으로 내려 넓이가 소멸하는 극한을 확인 → ② 풋/콜 스큐를 같은 값으로 두고 번갈아 올려 비교 → ③ 스큐를 마이너스로 내려 날개가 ATM보다 싸지는 역스큐 상태 확인. 마이너스 스큐는 OTM으로 갈수록 내재변동성이 낮아지는 상태 — 해당 날개의 곡선이 더 빨리 꺼지며 넓이가 줄고, VKOSPI가 ATM 변동성 아래로 내려갑니다. 스큐가 깊은 마이너스이고 ATM 변동성이 낮으면 날개 변동성이 0 바닥에 닿아 그 구간 옵션이 아예 무가치해집니다(곡선이 축에 붙는 지점). 같은 %p인데도 풋 스큐가 VKOSPI를 더 크게 밀어 올립니다 — 1/K² 가중이 낮은 행사가(풋 영역)를 증폭하기 때문이고, 이것이 ‘하락 공포에 더 민감한 지수’라는 성질의 수학적 원천입니다.

단순화: 0점이 정확히 떨어지도록 F=K₀=400으로 정렬해 보정항 (F/K₀−1)²=0인 설정입니다. 실제 산출에서는 F가 K₀보다 약간 위에 있어 보정항이 미세 조정 역할을 하며, 행사가 격자(ΔK=10)가 거칠수록 극저변동성 구간에서 이산합에 작은 잔차가 남습니다. 단일 만기(T=30/365) 근사로 30일 보간은 생략, r=3% 가정. 실제 KOSPI200 표면은 풋 스큐가 콜 스큐보다 훨씬 가파른 게 평상시 모습입니다.
